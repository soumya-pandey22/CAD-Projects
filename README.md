# CAD Projects 🔧

SolidWorks designs created as part of personal mechanical 
engineering projects.

## 🦾 Robotic Arm
- Designed in SolidWorks inspired by real industrial robots
- Pick and place mechanism
- Assembly of 7 parts: Base, Link 1, Link 2, 
  Link 3, Link 4, Support, and Gripper
- Each part individually modeled and assembled in SolidWorks

## 🛰️ Satellite

- Assembly includes: Satellite body, Solar panel, 
  Motor, and Motor Mount
- Inspired by real satellite design concepts
